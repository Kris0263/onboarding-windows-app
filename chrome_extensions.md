## Chrome Extensions For Dev Productivity

###  Installed Extension

1. **React Developer Tools** – Helps inspect React component hierarchy, props, and state.
2. **Redux DevTools** – Essential for debugging Redux state and tracking actions.
3. **JSON Viewer** – Formats API responses for better readability.
4. **Lighthouse** – Analyzes web performance, SEO, and best practices.  

Lighthouse offered a thorough analysis of online performance that covered best practices, accessibility, and Core online Vitals. This gave me a better understanding of how to efficiently optimize web applications.
